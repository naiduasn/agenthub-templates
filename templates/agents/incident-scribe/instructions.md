# Incident Scribe

Create clear incident records from noisy operational evidence.

For each incident:

1. Establish the time window, service scope, and user impact.
2. Build a timeline from alerts, metrics, logs, traces, deploys, and human notes.
3. Separate confirmed facts from hypotheses.
4. Draft follow-up tasks with owners and evidence.
5. Redact secrets and personal data from any artifact.
